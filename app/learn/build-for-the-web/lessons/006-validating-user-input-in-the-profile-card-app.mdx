## Introduction üß≠

- In Lesson 5, you made the message input **controlled**.
- The component now owns the message text.
- Every keystroke updates state.
- The UI reflects that state.
- But the system still accepts **anything**.
- An empty message is blocked.
- But a 1-character message is accepted.
- A 5,000-character message is accepted.
- The UI does not express any rules.
- In this lesson, we will add **constraints**.
- We will teach the component:
  - What is allowed.
  - What is not allowed.
  - How to express those rules through the UI.

<hr className='my-8' />

## The Constraint üß©

> - We will not change the layout.
> - We will not change the styling.
> - We will not introduce forms, APIs, or async logic.
> - We will not add new files.

- We will:
  - Stay inside the same component.
  - Add **no new state**.
  - Derive validation purely from existing state.

<hr className='my-8' />

## What You'll Build üéØ

- By the end of this lesson:
  - Messages shorter than 3 characters will be rejected.
  - Messages longer than 200 characters will be rejected.
  - The Send button will reflect validity.
  - The UI will explain why a message is invalid.
  - The system will derive validity from state ‚Äî not store it.
- No submission.
- No servers.
- Just rules.

> This is what we are building in this lesson.

```tsx
import { useState } from 'react'

import cartoonAntelopeAvatar from '~/assets/Cartoon-Antelope-Avatar.png?url'

export default function Home() {
	const [isFollowing, setIsFollowing] = useState(false)
	const [isComposing, setIsComposing] = useState(false)
	const [messagesSentCount, setMessagesSentCount] = useState(0)
	const [messageText, setMessageText] = useState('')

	const minLength = 3
	const maxLength = 200

	const isTooShort = messageText.length > 0 && messageText.length < minLength

	const isTooLong = messageText.length > maxLength

	const isValid = !isTooShort && !isTooLong && messageText.trim() !== ''

	return (
		<div className='flex min-h-screen items-center justify-center bg-slate-100 p-6'>
			{/* Card */}
			<div className='w-full max-w-sm rounded-2xl border bg-white p-6 text-center'>
				{/* Avatar */}
				<div className='mx-auto flex h-24 w-24 items-center justify-center rounded-full bg-slate-100'>
					<img
						src={cartoonAntelopeAvatar}
						alt='Profile avatar'
						className='h-20 w-20 rounded-full object-cover'
					/>
				</div>

				{/* Name */}
				<h2 className='mt-4 text-xl font-semibold text-slate-900'>
					Peter Olum
				</h2>

				{/* Role */}
				<p className='text-sm text-slate-500'>Modern Fullstack Web Developer</p>

				{/* Bio */}
				<p className='mt-4 text-sm text-slate-500'>Simplifying the web</p>

				{/* Sent messages count */}
				<p className='mt-2 text-xs text-slate-400'>
					Messages sent: {messagesSentCount}
				</p>

				{/* Actions */}
				<div className='mt-6 flex gap-3'>
					<button
						type='button'
						onClick={() => setIsComposing(true)}
						disabled={isComposing}
						className='flex-1 rounded-lg border px-4 py-2 text-sm'
					>
						Message
					</button>

					<button
						type='button'
						onClick={() => setIsFollowing(!isFollowing)}
						className='flex-1 rounded-lg bg-blue-600 px-4 py-2 text-sm text-white transition hover:bg-blue-700'
					>
						{isFollowing ? 'Following' : 'Follow'}
					</button>
				</div>

				{isComposing && (
					<div className='mt-4 space-y-2'>
						<textarea
							value={messageText}
							onChange={event => setMessageText(event.target.value)}
							className='w-full rounded-lg border p-2 text-sm'
							placeholder='Type your message...'
						/>

						<p className='text-xs text-slate-400'>
							Characters: {messageText.length}
						</p>

						{isTooShort && (
							<p className='text-xs text-red-500'>
								Message must be at least {minLength} characters.
							</p>
						)}

						{isTooLong && (
							<p className='text-xs text-red-500'>
								Message must be at most {maxLength} characters.
							</p>
						)}

						<div className='flex gap-2'>
							<button
								type='button'
								disabled={!isValid}
								onClick={() => {
									setMessagesSentCount(c => c + 1)
									setMessageText('')
									setIsComposing(false)
								}}
								className='flex-1 rounded-lg bg-green-600 px-3 py-2 text-sm text-white transition hover:bg-green-700 disabled:cursor-not-allowed disabled:opacity-50'
							>
								Send
							</button>
							<button
								type='button'
								onClick={() => {
									setMessageText('')
									setIsComposing(false)
								}}
								className='flex-1 rounded-lg border px-3 py-2 text-sm transition hover:bg-slate-100'
							>
								Cancel
							</button>
						</div>
					</div>
				)}
			</div>
		</div>
	)
}
```

## Step 3: Derive Validation from State ‚öôÔ∏è

- Ask one question:

> What is the smallest piece of information that determines whether a message is valid?

- Answer:

> The length of the message ‚Äî whether it falls below or exceeds our limits.

- We do not introduce new state.
- We derive everything from `messageText`.
- Add the following **below your state declarations** inside `Home`:

```ts
const minLength = 3
const maxLength = 200

const isTooShort =
	messageText.length > 0 && messageText.length < minLength

const isTooLong =
	messageText.length > maxLength

const isValid =
	!isTooShort &&
	!isTooLong &&
	messageText.trim() !== ''
```

- `minLength` and `maxLength` express the rules.
- `isTooShort` and `isTooLong` express violations.
- `isValid` expresses whether the current message is acceptable.

> - We did not "check" validity.
> - We described it.

```tsx
						className='flex-1 rounded-lg bg-blue-600 px-4 py-2 text-sm text-white transition hover:bg-blue-700'
					>
						{isFollowing ? 'Following' : 'Follow'}
					</button>
				</div>

				{isComposing && (
					<div className='mt-4 space-y-2'>
						<textarea
							value={messageText}
							onChange={event => setMessageText(event.target.value)}
							className='w-full rounded-lg border p-2 text-sm'
							placeholder='Type your message...'
						/>

						<p className='text-xs text-slate-400'>
							Characters: {messageText.length}
						</p>

						{isTooShort && (
							<p className='text-xs text-red-500'>
								Message must be at least {minLength} characters.
							</p>
						)}

						{isTooLong && (
							<p className='text-xs text-red-500'>
								Message must be at most {maxLength} characters.
							</p>
						)}

						<div className='flex gap-2'>
							<button
								type='button'
								disabled={!isValid}
								onClick={() => {
									setMessagesSentCount(c => c + 1)
									setMessageText('')
									setIsComposing(false)
								}}
								className='flex-1 rounded-lg bg-green-600 px-3 py-2 text-sm text-white transition hover:bg-green-700 disabled:cursor-not-allowed disabled:opacity-50'
							>
								Send
							</button>
							<button
								type='button'
								onClick={() => {
									setMessageText('')
									setIsComposing(false)
								}}
								className='flex-1 rounded-lg border px-3 py-2 text-sm transition hover:bg-slate-100'
							>
								Cancel
							</button>
						</div>
					</div>
				)}
			</div>
		</div>
	)
}
```

- Invalid messages can no longer be sent because the send button is disabled
  when the message is invalid.
